# Deployment Addresses

This directory contains deployment addresses for each network.

Files are automatically generated by deployment scripts in the format:
- `{chainId}.json` - Core protocol deployment
- `avs-{chainId}.json` - AVS contracts deployment

## Networks

- Base Sepolia: 84532
- Base Mainnet: 8453
- Ethereum Sepolia: 11155111
- Ethereum Mainnet: 1

## Usage

Deployment addresses are saved after running:
```bash
forge script script/DeployBastion.s.sol --broadcast
```

Example deployment file structure:
```json
{
  "poolManager": "0x...",
  "bastionHook": "0x...",
  "insuranceTranche": "0x...",
  "lendingModule": "0x...",
  "bastionVault": "0x...",
  "volatilityOracle": "0x...",
  "chainId": 84532
}
```
